sudoku-solver
=============

Sudoku solver written for a [InterviewStreet][IS]
[CodeSprint][CS]. The solver uses backtracking search and forward checking for filtering. The code is light on comments and has some rough edges, keep in mind the CodeSprint had a 12 hour deadline.

[IS]: http://www.interviewstreet.com 
[CS]: http://sudoku.interviewstreet.com
